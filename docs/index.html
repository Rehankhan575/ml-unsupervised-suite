ML Portfolio Project

Customer Segmentation
with RFM & K-Means

A production-grade unsupervised learning system trained on 397,884 UK e-commerce transactions. Segments customers into four behavioral groups using RFM feature engineering and K-Means clustering, served via a FastAPI backend containerized with Docker.
